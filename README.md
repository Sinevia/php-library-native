## Installation ##

Add the following to your composer file:

```json
   "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/sinevia/php-library-native.git"
        }
    ],
    "require": {
        "sinevia/php-library-native": "dev-master"
    },
```
